# Database Integration Implementation Plan

## Goal
Integrate a PostgreSQL database to persistently store all triggered alerts generated by both the Chat Analyzer (Part 1) and Video Analyzer (Part 2), while also improving Part 1 to explicitly display a numerical Confidence Score alongside the reason for the alert.

## Proposed Changes

### Component 1: Database Management (New)
#### [NEW] [db_manager.py](file:///c:/Users/atsha/Downloads/melodywings/db_manager.py)
- Create a reusable PostgreSQL connection wrapper using `psycopg2-binary`.
- Use the credentials provided (`host=localhost`, `port=5432`, `dbname=postgres`, `user=postgres`, `password=atshu@10`).
- Create an initialization function that creates a `safety_alerts` table if it does not exist. 
- Table schema: 
  - `id` (SERIAL PRIMARY KEY)
  - `timestamp` (TIMESTAMP)
  - `alert_type` (VARCHAR) - e.g., "GROOMING", "TOXICITY", "PII_PHONE"
  - `trigger_text` (TEXT)
  - `confidence_score` (FLOAT)
  - `reason` (TEXT)
